## Kontribusi Dokumentasi

Saat menambahkan dokumentasi baru:

1. **Gunakan format Markdown (.md)**
2. **Struktur yang jelas** dengan heading dan subheading
3. **Sertakan contoh kode** jika relevan
4. **Tambahkan screenshot/diagram** untuk penjelasan visual
5. **Update index** di README.md terkait
